Чтобы бот работал корректно нужно создать файл .env в директории программы.
Файл должен содержать

ACCESS_TOKEN="ваш токен" 

DATABASE = "название вашей базы данных"

USR = "юзернейм вашей базы"

PASSWORD = "пароль к вашему юзеру"

HOST = "ваш хост"

port = "ваш номер порта"

Может ругаться на пакет telebot тк устанваливается PyTelegramBotApi
